# Interactive Demos

- [Fibonacci](fibonacci.md) — recursive sequence generation
- [Merge Sort](merge_sort.md) — classic divide-and-conquer sorting
- [Binary Search Tree](binary_search_tree.md) — insertion and lookup in a recursive ADT
- [Expression Evaluator](expression_evaluator.md) — evaluate a tiny arithmetic AST
- [Dijkstra Lite](dijkstra_lite.md) — shortest-path relaxation on a tiny weighted graph
- [0/1 Knapsack](knapsack_01.md) — dynamic-programming row updates
- [Union-Find](union_find.md) — disjoint-set connectivity on a small graph
- [Prefix Parser + Evaluator](prefix_parser.md) — recursive-descent parsing of prefix tokens
- [Topological Sort](topological_sort.md) — Kahn-style ordering on a DAG
- [N-Queens](n_queens.md) — backtracking with diagonal pruning
